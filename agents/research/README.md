# ResearchAgent

Specialized agent module for performing web search, literature research, document synthesis, and information extraction.

## Files
- `agent.py`: `ResearchAgent` class extending `BaseAgent`.
- `config.py`: `ResearchAgentConfig` pydantic settings.
- `memory.py`: `ResearchAgentMemory` adapter.
- `prompts.py`: Prompt definitions.
- `tools.py`: Tools registered for research tasks.
