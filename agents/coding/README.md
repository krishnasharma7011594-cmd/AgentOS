# CodingAgent

Specialized agent module for automated software architecture, code generation, refactoring, linting, and unit testing.

## Files
- `agent.py`: `CodingAgent` class extending `BaseAgent`.
- `config.py`: `CodingAgentConfig` settings.
- `memory.py`: `CodingAgentMemory` adapter.
- `prompts.py`: Prompt templates.
- `tools.py`: Tools for code manipulation and execution.
