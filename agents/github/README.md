# GitHubAgent

Specialized agent module for managing GitHub repositories, pull requests, issue tracking, and automated releases.

## Files
- `agent.py`: `GitHubAgent` class extending `BaseAgent`.
- `config.py`: `GitHubAgentConfig` settings.
- `memory.py`: `GitHubAgentMemory` adapter.
- `prompts.py`: Prompt templates.
- `tools.py`: GitHub API tools.
