# FinanceAgent

Specialized agent module for processing financial data, quantitative modeling, market analysis, and summary generation.

## Files
- `agent.py`: `FinanceAgent` class extending `BaseAgent`.
- `config.py`: `FinanceAgentConfig` settings.
- `memory.py`: `FinanceAgentMemory` adapter.
- `prompts.py`: Prompt definitions.
- `tools.py`: Financial tools registry.
